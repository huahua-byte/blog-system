# 快速开始

这篇文章演示一篇标准公开文章应当如何编写 frontmatter 和正文内容。

## 第一步

在 `docs/posts/` 下创建新的 Markdown 文件，并将 frontmatter 放在文件最顶部。

## 第二步

为文章补齐分类、标签、摘要和封面路径，保持字段顺序统一，便于维护和检查。

## 第三步

运行 `npm run dev` 或 `npm run build`，确认 VitePress 可以正常解析页面。
